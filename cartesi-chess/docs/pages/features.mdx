# Some of the tech stack we used while building Cartesi.mef

## 1. Sign in with Ethereum (SIWE)

The SIWE authentication model is used to confirm the person interacting with the dapp is the one using the wallet address provided. So users have to confirm their wallet before even having access to the game and only those with the token bridged will be able to play the game

## 2. Web Socket 

The Web Socket protocol which layers on the http protocol is used to ensure a real-time gaming experience between the players
